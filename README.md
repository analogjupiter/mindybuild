# mindy-build

**mindy-build** is an open-source build configuration and build system.

## Notable features

- Native support for the D programming language.
- Elegant, descriptive build recipies.
- Optional **zero-configuration builds**.
- Two-stage system: Build Configuration System + Build System.

## Building

- Run `./build.sh` on POSIX.
- Run `.\build.cmd` on Windows.
- Or use DUB.

The resulting executable binary can be found in the subfolder `bin`.
